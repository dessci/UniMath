Each equation is deliberately followed by some punctuation B, so the spacing after the equation is shown, C.
The following equation is an example of an inline expression, which is split into several characters: DEFGH. This makes it possible, using the CSS property word-wrap: break-word, to break an inline equation at specific points, if the browser has no other option.
There is also a zero-width-space character &#8203; which leads to D​E​F​G​H, not sure about the browser support, though.
Tall expressions may extend into the previous and/or following line, like this one J.
This can be avoided by rendering the expression at, e.g., 25% of its normal size and then using font-size: 400% in the following way: I (possibly also adjusting the baseline).
